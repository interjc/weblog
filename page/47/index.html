Rec & Cloverfield

发表于 2008-04-28 更新于 2020-04-19 分类于 琐事 Disqus：

前两天看了两部好玩的片子，Rec[死亡录像]和Cloverfield[柯洛弗档案]。 先说Cloverfield，这是一部DV片，讲述的是纽约突然遭到奇怪生物攻击，一群年轻人在里面乱蹿的故事，整个故事结构还算紧凑吧，值得一看。当然你可不要怀着看美国大片的心情去看，那样你会失望的。 片中有个场景挺有趣的，你看了电影也许这张图片可以给你一些启发哦(图片比较大，请耐心等待，或者你可以下载他慢慢看)： 然后说死亡录像，这个片子有人说是DV片，我觉得不应该这么叫，毕竟别人用来拍摄的是电视台专用的摄影机了。不大喜欢那个西班牙女主持，当不是很多人所说的喜欢尖叫，而是这个人话太多不大懂事，一下这张海报里面就是她的尊荣(你仔细看片可以知道是哪一幕)，不过，当然，她大部分时间也还没有这么恐怖了，毕竟还是有观众喜欢她的： 觉得片中扛摄像机的那位能力挺强的比较佩服，只是从头到尾也没有露过脸实在遗憾，有位后来拿锤的消防队员也颇有风范，让我依稀想起山丘之王的音容笑貌，还好这位露了脸。 死亡录像这部片终于让我时隔多年之后找到了看恐怖片的感觉，实属不易，建议对恐怖片已经麻木的同学们试试，也许会有效。至于剧情嘛，其实设计的也是很精妙的，如果你胆子够大可以一个人静心温习几遍回味几次，呵呵。 现在这两部应该属于比较热的片子了，所以我也不提供BT种子或者下载地址了，想看的自己随便找找都是一大片的，这里仅提供一个Rec的中文字幕(VeryCD里自带那个字幕是英文的)：下载 PS：胆小或吹毛求疵者慎看。

VirtualBox

发表于 2008-04-24 更新于 2020-04-19 分类于 pc ， Tools Disqus：

收到Ubuntu7.10 Gutsy光盘是在半年以前，由于之前并没有脱离Windows而使用Linux的完整经历，所以我不大敢只装一个Ubuntu，装双系统麻烦更多，所以在虚拟机上安装Ubuntu成了最佳选择。 寻找一个合适的虚拟机是当务之急。之前也用过VMware，可他的问题实在是很多，体积又庞大，那个精简版bug不少不说，一个网络连接都不是轻易可以搞定的，总之一个字，闹心。直到发现了VirtualBox，安装并使用了VirtualBox大约10分钟后，我的第一感觉是，我可以去把我的VMware删除了，于是我就这么做了。 VirtualBox是由德国 InnoTek 这家软体公司出品的软件(已经被疯狂的sun收购了)，它提供使用者在 32 位的 Windows 及 Linux 作业系统上虚拟另外的机器执行其它 x86 的操作系统，使用者可以在 VirtualBox 上执行 Windows、Linux、OS/2 Warp、OpenBSD 及 FreeBSD 等系统作为客户机操作系统。在 2007 年 1 月，InnoTek 以 GNU General Public License (GPL) 释出 VirtualBox，并提供二进位版本及开放源码版本的代码。VirtualBox 可以在 Linux 和 Windows 主机中运行，并支持在其中安装 Windows (NT 4.0、2000、XP、Server 2003、Vista)、DOS/Windows 3.x、Linux (2.4 和 2.6)、OpenBSD 等系列的客户操作系统。 这款软件的特色是：免费、开源、多国语言版、原生支持许多硬件，不用再装N块虚拟网卡和N个不知所谓的协议了，我在里面重装了我的Ubuntu，所有以前存在的问题现在都没有了，在无需任何设置的情况下只要宿主机器可以上网虚拟机就可以上网，目前我已经不小心升级到了8.04 Hardy了。 你也想删除你那VMware投入VirtualBox的怀抱了吗？到这里下载吧，目前最新版本是1.5.6：官方、华军、其他
