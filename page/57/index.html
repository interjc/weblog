电脑

发表于 2007-12-21 更新于 2020-04-19 分类于 琐事 Disqus：

当我今年初拖着几个硕大的行李箱坐着小的仅供停放身体的卧铺火车前往深圳的时候，我就明白，想要把那台陪自己征战了3年有许的电脑随身携带基本上是完全不可能了..于是我琢磨着，过去了该买台新机器了。由于种种客观和主观原因，这电脑一直没有买，心说一年也就这么个愿望，总不能都快圣诞节了都没有一件事办成吧，于是乎上周，终于买了这台电脑，虽然配置不算太高，但是总算可以在自己的机器上写blog、开支付宝和静下心来学习点东西了..经过一个月左右的研究，本着创办节俭社会的原则，还是选择了AMD 4000+和MCP68的组合，显示器选择了5000：1对比度的L194WT，用了几天下来总体感觉还是不错的，比家里那台强劲不少(囧..废话..)，唯一的遗憾是这显示器是LG的，违背了之前抵制韩货的原则的原则。 买了电脑当然要上网，不然那就是一坨废铁，所以马上去申请电信的ADSL，电信真厉害，只要你报出银行卡号和身份证号，他就可以从你帐户里面扣款了，国营垄断企业就是与众不同，不过价格还比较合理，1M才￥80/月。电信效率还挺快，没过两天，他的短信就来了：由于您所在的区域没有主干网，所以您的申请被取消，农历年后有可能新建机房.. 农历新年，那我还不得等到花儿也变成肥料了啊～ 遂赶快寻找紧急替代措施，没想到这一找不要紧，噩梦就这样随之开始了。替代品是这城中村房东搞的网络共享，打电话的时候他说无限时无限制光纤接入上网，装的时候再问，他说是3M总带宽，还要十几个人来合用，我的天啊，3M带宽我自己用还凑合，还要合用？！后来没办法，总不能买个电脑当影碟机吧，于是乎装了。首付150大洋3个月的网费，我想这厮估计也就是做这种三个月生意了，万般无奈我装了。这厮装模作样的设置了一下路由器，我发现居然用的是TP-LINK，感觉天空的乌云又多了一些，OKOKOK，我是冤大头我认宰，只要我装了可以用再慢我也认了..我问他是不是什么都可以，他说可以，我问可不可以下BT，他说不行，这时候他说了一句我也许会记3个月的话：BT是变态下载，他是变态的.. 我说那我的电影从哪来？他说可以买碟.. 我说那我下歌怎么办？他说你可以用迅雷。我心说BT不用我就用迅雷吧，也是可以下一些电影的，然后他就走了。由于这时候我的电脑还是裸机，所以要先下点应用软件什么的，我就用迅雷下载了，速度还挺快，接近180，心里没那么囧了，可是好景不长，过不了多久我发现网页打不开了，迅雷也定住了，打电话一问，曰：马上开通。过不多久可以上了，再下，又断网，再打，曰：好，马上开通。就这样一个小时停了5次以上，什么好心情都没有了，连windows更新都无法完成，再一次的断网以后，我再打，发现关机了，这时候我有了想要K他的强烈愿望。又一个小时过去了，我和mumu通过观看从手机里拷到电脑的3GP打发时间，接着再试那个电话，通了，我问：你们这什么破网？我用迅雷你也封？？回答依然是：好，马上开通。然后就挂了。之后一直不敢下载，相安无事，过了一会儿，需要下一个软件，无意间打开了迅雷，瞬间又断网，这一次停了一晚上，那头电话也是，第二天中午打通那边电话，曰：你的迅雷也要限速，而且不要太大。我就问：NTMD你就说你这个破网到底可以干什么？你说说看我最大下载速度到底是多少？你是要我每天都只聊QQ看网页打打斗地主吗？你以为我真是你爹啊？那头曰：好，马上开通！全世界人民震惊了，这个世界上怎么会有这样的2逼。 我想起一个笑话，某人就要死了，面临去天堂和去地狱的选择，天堂客满，要去就得等，这时他看见了地狱的一个画面，美女洋房，沙滩阳光，于是二话不说去了地狱，可到了那里却只看见几个胡渣男在搅油锅，这人上前一问，原来他先前看到的是广告。原来这种事不知发生在笑话里，他还发生在我的身上，而且比那个更糟，我是明知那是地狱，可是组织上没有给我去天堂的指标。 在这里我想告诉那些在城中村居住的同仁们，有条件还是去拉一个人一条线的电信ADSL吧，那才是天堂，不管你怎么骂，永远不会随便断你线的却只有电信。 请电信聘请的民工们过年赶快回来挥舞你们的工具，赶快给我造一个新的机房吧～ 我是刚买电脑的在深圳关内一个暂时没有电信宽带线路资源的城中村租房住的interjc，这是我的故事.. to be continued..

谷歌

发表于 2007-11-07 更新于 2020-04-19 分类于 IT Disqus：

谷歌最近动作频频，G.cn上线了，整合了天气预报功能，Gmail全面升级，改进了Adsense，OpenSocial和Gphone的真身Android即将上马，让我们对Google充满了期待。现在Google的中国团队谷歌又为我们带来了一个作品，只是它略显搞笑，他就是http://www.google.cn/hp?sp=china，这样一个东西让我想起了Google导航，也让我想起了那个做了SEO的Google，难道李开复真的想要那样么？我随便看了看，这个网站整个界面做的还是比较和谐的，界面亲和力不错，可仔细一瞧也不过是众多搜索结果的一个动态整合以及半个好123，这样的一个东西真的可以从百度那里抢来那一部分电脑知识不够的网民吗？我觉得有点悬..这样的网站只能说明谷歌团队不够坚决，想要学百度走大众化路线却始终放不下自己高尚的架子，恕不知，不作恶在中国只有吃亏的分.. 不得不承认，Google中国最近有点心浮气躁了，虽然他们最近取得了一个巨大的公关成果，不过这都不能掩盖其在用户(特别是初级网民)体验上的不足。最近由于公司网络只能上Google和百度以及部分门户网站，所以我的上网范围被缩得很小，这让我发现了除了GReader和GTalk以外，我使用的最多的竟然是曾经BS的百度，我上贴吧聊天(Google Groups基本没有人)，我在知道里搜索一些自己不想花时间找的问题(Google有类似功能不过我想很少有人用过)，我在百度掌门那里看现在流行什么歌曲并很快找到下载链接和即时与有同类爱好的人交流(别提Google的音乐趋势了)，我用百度空间来复制粘贴我看到的想要完整保存下来的图片和文章(blogger也可以用，可是又有多少人可以持续看得到呢？)，猛然间我发现自己竟然如此百度，对于我这个GFan来说真的是很讽刺了。Google中国(实在不想称你为谷歌，因为实在不喜欢这个名字)，你什么时候可以好用一点呢？不要再那么小众了好吗？ 匆忙下笔，不知所云..各位将就看看吧～
