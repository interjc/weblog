死神(Bleach)后续剧情的猜想

发表于 2009-07-15 更新于 2020-04-19 分类于 Fun Disqus：

Bleach现在又一次进入了插科打诨的枯竭期，等一个多星期上土豆一看，又是闹眼子，实在是令人失望。 其实细想下来，这也不怪制作组，因为一开始这个动漫的主要思想局限性太大，就是一护一行人上静灵廷与护廷十三队争夺伙伴的故事，可后来发现这个打完了的话动漫就玩完了，于是乎觉得不行，就设下了蓝染这另一条主线，现在护廷十三队和蓝染的决斗正在进行，不管剧情如何拖沓，这场仗也都有完结的一天，不管他们的胜负关系如何，如果动漫想要继续进行下去，就必须有新的剧情来发展。 新的剧情有几个必要的前提： 前提1：主角一护一行人和队长中最受喜爱的人物不能死； 前提2：剧情必须可持续的发展下去； 前提3：不能落入俗套。 下面是我的猜想： 猜想1：在虚圈外的护廷十三队全灭，蓝染掌控了尸魂界，摧毁了空座市，建立了王键，总之就是实力得到了巨大提升，虚圈内的一护一行人击败那个最装精的名字我忘了的那个十刃之后对蓝染进行反攻。 这样的话，剧情就会落入俗套，而且最大的BOSS变成了蓝染，蓝染一死就没戏了，蓝染不死久而久之就成老妖怪了。最重要的是很多备受大量粉丝用户的队长们都死了，会引起这些人的不满。 猜想2：在虚圈外的护廷十三队被打残，一护众人通过某些特殊渠道赶来救场，打败了蓝染，这时候有人来接条，又将一护众人和被打残的队长们打趴下，这时候大家才发现原来蓝染只是一颗棋子，背后还有更大的阴谋，护廷十三队也只是整个系统中代表正义的最基层的组织而已。 如果这样的话可以大大丰富剧情可以发展的线路，各位队长也有了更大的发展空间，也可以插入更多的人物。不过这个阴谋要设计的够巧妙。 猜想3：在虚圈外的护廷十三队和蓝染的争斗打得两败俱伤，这时候一护一行人赶到了，于是乎蓝染众人就走为上计了，飙一句我还会再来的，之后就是我们在魔兽RPG防守型地图中常见的模式了，一护一行人和护廷十三队守护着各自的地方，防止一次又一次变得更强的蓝染众人的进犯。 这种剧情貌似比较无脑，但是也不失为一种办法，至少以后的剧情有着落了，只要开发出更多的招式来就OK了，可谓绝佳的编剧养老型剧情。 猜想4：所有已经出现过的角色混在一起捉对厮杀，这时候有一股更强大的力量出现了，瞬间秒了其中的一方，一般说来是邪恶的一方，这也就是最后的BOSS，也有可能是护廷十三队的上司，被秒的可能正是护廷十三队，一护等人被神秘人物解救，估计是浦原，然后这个争斗就变成了另一种原因另一种形式的争斗，说不定蓝染会和幸存的一护一行人联合起来对付新的BOSS。 这样的话，死神的个动漫也就变成了真正的动漫版无间道了，蓝染到处当卧底，正邪不分，不过这也可以迎合某些大叔控的口味吧.. 猜想5：一护直接出现秒掉了所有的人，原来他体内有一个更大的精神力量，已经对一护全面的进行了控制，所以他实力爆棚了，然后他就成了暂时的最大的BOSS，这时候还出现了不少神秘人来成为一护的新手下。幸存的死神们开始了和一护势力对抗的新历程，然后一护的真实身份也慢慢揭晓，其实他也只是被幕后的黑手控制了而已，于是乎大家的目标又变成了解救一护，打败幕后黑手。 这样设置的话，把主角给搞没了，变成了反面人物，不知道粉丝们能不能接受了。 本来还有的，但是我键盘坏了（回车键按下去弹不起来），实在影响心情，所以后面的想不起来了，以后想到再补吧。 PS：目前还在更新的三个主要的老牌动漫，我在看其中两个：One Piece和Bleach。One Piece这个更老牌一点，剧情嘛也是行云流水有无限发展的可能，这里就不说了，顺其自然吧。 欢迎拍砖，谢绝人身攻击！

豆瓣电台

发表于 2009-07-03 更新于 2020-04-19 分类于 Fun Disqus：

网络爆炸的年代，我们获取歌曲的方式越来越多，方法越来越简单，我们的存储设备越来越大，我们存下来的歌越来越多，多到我们需要搜索才可以找到自己的某一首歌了。我发现，自己的歌收集得越来越多的同时，自己已经不知道听什么歌，不知道怎么听歌了。 一直在寻找那种感觉，我没法选择，我只是等待，等待别人放给我听的歌正好我喜欢，我不知道有多少人有我这样的想法，豆瓣电台显然知道有我这样的人存在，他的这个电台非常符合我的胃口～ 简洁的界面，只有收藏、下一曲、永远略过和音量这几个最基本的功能，让我们享受那音乐的纯粹..真的是爱不释手啊～～ PS：非常感谢@roamlog的邀请，让我终于用上了这么一个好东东～

写在看完变形金刚II之后

发表于 2009-06-29 更新于 2020-04-19 分类于 Fun Disqus：

变形金刚II(Transformers:ROF)这样一部电影，从我从电影院看完他的第一部就开始期待了，昨天终于有幸去电影院看了。 画面很，庞大，震撼说不上，可能在1的时候已经给我震完了吧。影院的效果就是好，所以看这个电影确实是一种享受的。当然就我个人看来他想超越1或者是原版动漫是没有多大可能了。 我不是一个喜欢搞剧透的人，所以我非常不想说剧情。 只说一下对于剧情的感受： 1. 剧情太商业化，变形金刚这么强大的战斗力和防御力，我不知道拿着枪的人类可以对他们造成什么样的威胁呢？美国大兵们与变形金刚们短兵相接，难道是为了方便狂派们刷数据么？ 2. 由于是美国电影，所以一定要展现美国军备的强大，所以战舰上的秘密武器可以KO看上去甚是强大的纸老虎——狂派合体机器人； 3. 主角一定得是人类，为了烘托人类主角的伟大性，不惜牺牲同样伟大但没有他伟大的各位领袖同志，主角由于起点比较低，所以随便摸一下某个能量体就可以吸收里面的精髓； 4. 赶新潮，年轻人做网站办公司、经济危机，再在电影里融入一点青春元素，把学生宿舍比喻为霍格沃茨，可惜这方面的篇幅太短，如果开发一下说不定会为本集贫乏的剧情添加一点色彩； 5. 冷兵器，变形金刚们之间的斗争，如果想要解决对方，就必须使用冷兵器或者蛮力，这方面是我所欣赏的，我可不希望擎天柱、威震天是被一把麦林爆头干掉的； 6. HappyEnding，每一部想拍续集的电影都有那么一个HappyEnding，狂派还会回来的誓言也是必需的～ 综上所述，这的确是一部好电影，看的时候请放松您的大脑，因为没什么可以让你去思考的。 BTW：搬到cosbeta的主机以后速度很快，很快，我非常欣慰。
