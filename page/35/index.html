SSH

发表于 2009-07-31 更新于 2020-04-19 分类于 Web Disqus：

今天让我们来介绍一下SSH，首先看看维基百科的解释：

SSH 为 Secure Shell 的缩写，由 IETF 的网络工作小组（Network Working Group）所制定；SSH 为建立在应用层和传输层基础上的安全协议。 传统的网络服务程序，如FTP、POP和Telnet其本质上都是不安全的；因为它们在网络上用明文传送数据、用户帐号和用户口令，很容易受到中间人（man-in-the-middle）攻击方式的攻击。就是存在另一个人或者一台机器冒充真正的服务器接收用户传给服务器的数据，然后再冒充用户把数据传给真正的服务器。 而 SSH 是目前较可靠，专为远程登录会话和其他网络服务提供安全性的协议。利用 SSH 协议可以有效防止远程管理过程中的信息泄露问题。透过 SSH 可以对所有传输的数据进行加密，也能够防止 DNS 欺骗和 IP 欺骗。 SSH 之另一项优点为其传输的数据是经过压缩的，所以可以加快传输的速度。SSH有很多功能，它既可以代替 Telnet，又可以为FTP、POP、甚至为 PPP 提供一个安全的“通道”。

简单的说起来，SSH就是提供给你一个安全的网络通道，这样你的个人信息在浏览信息的同时就不会被他人获取，保障了你的信息安全；其次，由于中国大陆的网络公司之间互相扯皮，而且互联网出口带宽有限，经常出现网络异常的状况，所以这个时候通过SSH这种中间转换的方式来上网甚至会更快一点。 blockcn.com提供了30元一年的SSH出售，还算是比较便宜了，有兴趣的可以去试试。让我们来共同学习SSH这种新兴的网络知识。 PS：SSH能提供安全的通道这不假，但网络上的言论鱼龙混杂，真伪难辨，你所接触到的内容未必也都安全，请大家上网的时候一定要注意甄别，以防上当受骗。

我叫MT

发表于 2009-07-30 更新于 2020-04-19 分类于 Fun Disqus：

以下是目前人气最火，好评最多，民间自创，无比牛逼，85后、魔兽盲一般看不懂却也可以看的魔兽题材国产动画，相当的经典，目前才出到第二集，严重期待第三集， 已更新至第十集， 啥也不说，先看动画： 记得刚开始玩WoW这个游戏的时候是某年五一节，之前就对魔兽争霸每日几练的我，自然会对这款大作充满了期待，内测资格没有，却也通过各种渠道预先学习了许多，就等公测开服。公测不要钱，而且大家都是怀着一颗崇敬的心去玩的，虽然总是有N个人排队，有时候早上起来就排队，中午吃饭回来还没排上，差点放弃了，可自从学会了秒排，什么也阻止不了我们前进了。 开服玩的有且只有一个的职业是暗夜小德，从1-60级都是用恢复天赋打上去的，生活之艰辛可想而知。 我记得刚开始玩的时候在蜘蛛洞就有人和你组队一起做任务了，大家都很客气谦虚和谦让，打出什么装备生怕别人吃亏了，很有点孔融让梨的意思； 自己做的最复杂的任务是16级单枪匹马去做变海豹的任务，由于不认识路，从米奈希尔港沿着海岸线游到了西部荒野，整个任务打了一晚上，完成时那叫一个高兴啊，到处找小河沟边来给别人看； 打的第一个副本是黑暗深渊，印象最深的却是死亡矿井，因为我记得那天打到一半灭团数次我突然上网卡没钱掉线了，后来我连带下楼买卡到充值耽误了20-30分钟左右，回去时发现伙伴们还在，都坐那儿聊天等我在，这在今天的WoW基本上是无法想象的； 第一次大型PVP是20多级在灰谷，由于联盟部落抢地盘，最后我们这些20-30级的小盆友就由几个45级的大哥（当时45级封顶）带领，组成几个团，集体杀到了十字路口，我的回忆是——幻灯片，没错，那时候我还是256M内存条呢； 第一个游戏里的朋友是叫做Dolimi的术士，记得有一天我在黑海岸打怪，见地上有个尸体，我上去顶着怪给他复活以后自己死了，从此以后他只要和我一起玩就会把灵魂石绑在我身上，后来又遇见了叫兔兔我爱你的法师，想必他的女朋友就叫兔兔吧，这人是在他免费派送包包的时候认识的； 加的第一个工会叫什么我忘了，只记得会长是我在荆棘谷打怪时遇到的一个叫做爱太难的精灵战士，老是对坐着正在喝水的我说，“我上了”，于是以迅雷不及掩耳盗铃之势他倒在了怪堆里……； 当时我感觉自己真的是在玩游戏，后来由于种种原因，朋友们一个个消失了，直到自己也终于60级了，由于无法忍受上班一样的MC，渐渐的没有玩了。 后来总有忍不住上去看看的时候，发现好友全都灰了，工会也不复存在，想组个野队下副本体验一下，对方伸手就开始要钱了，我们当年和谐欢快的WoW，什么时候变成了这个样子？可现实是，再美好的童年，也只能封存在回忆里，再也无法复刻了，感谢暴雪和9c给我们带来的曾经的快乐。也谢谢这部动漫让我能将美好的日子再重温一遍，祈祷千万不要太监了。 BTW：有能制作《我叫MT》这样优秀动漫的人才，国家为什么还要推广弱智的喜洋洋？

CSS Viewer For Firefox 3.5.x

发表于 2009-07-27 更新于 2020-04-19 分类于 Tools Disqus：

Firefox 3.5.x出来了这么久了，CSS Viewer依然没有针对其进行更新。作为一个我制作网页时必用的Firefox插件，他一度让我打消了从3.0.x升级到3.5的念头，不过Firefox 3.5系列对于启动速度的优化实在诱人，现在终于让我找到了这个既可以使用CSS Viewer插件，又可以享受FF 3.5.x高速启动的办法。 1.下载CSS Viewer的本地安装文件(xpi文件)，请右键另存为； 2.用解压缩文件打开这个文件； 3.将其中的 install.rdf 拖出来用记事本或者其他任意文本浏览器打开； 4.将其第18行中的3.0.*改为3.5.x，保存； 5.将此修改过的 install.rdf 拖回压缩包保存； 6.安装修改后的 xpi 文件(将此文件拖到FF窗口安装即可)； 这样就大功告成了，如果你不喜欢自己去做这些麻烦事，大可以下载我帮你修改好的，地址如下： Google Code | Clip2Net | 大米盘 BTW：CSS Viewer制作组可真够懒的，看见可以用就懒的修改了，还要咱们自己来改。也有人说用什么nightly插件可以解决兼容性问题，可是那个插件我装了一下，实在不知所云也就懒得研究了，反正FF对我而言也只是一个网页开发工具而已。
