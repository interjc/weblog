电话实名

发表于 2007-09-26 更新于 2020-04-19 分类于 ADs Disqus：

多年前网络上曾经出现过一种很有创意的东西，他的名字叫做网络实名，那时候人们对于网络不太熟悉，需要类似的东西来辅助上网(特指那些不太会上网的群众，而这些人又恰恰是网络实名最潜在的用户)，后来由于事实证明这确实很有市场价值，于是各路神仙齐上阵，其沦为流氓的代名词。 现在，又一种实名服务产生了，他就是电话实名。 其官方介绍是这样的：

“中国总机电话实名”是以企业的“公司名称”、“品牌”、 “商标”、 “产品名称”、“行业名称”等作为标识的一种现代化的中文通信方式。用户只需记住企业的“电话实名”，拨打4008100100，转接企业的“电话实名”，即可与企业通话。所谓“记得住 找得到！”而且只收市话费。 “电话实名”符合中国人的记忆、联想、思考习惯，考虑到人对文字的记忆远大于对数字的记忆。这样，人们再也不用记忆各种冗长的固定电话、手机或小灵通号码，而只要直接使用一个简单易记的中文名称，就能直接联系到目标企业。 “电话实名”以其方便快速、简单易记的特点被广泛应用于企业营销、产品促销、品牌推广等方面，使企业商机无时不有、无处不在。同时，企业注册“电话实名”，也是企业知识产权保护的必要手段。

和网络实名不同的是，这种服务是建立在电话上的，通过拨打4008100100这个电话你就可以转接到你要找的企业去了，个人感觉有点像号码百事通，对于那些比较喜欢使用传统方式来寻找服务的用户比较有用。 这个市场的前景还是可观的，连谷歌都在进行类似的服务，只不过，竞争对手的实力貌似更强大一些，毕竟114已经是一个很老很老的品牌了，况且这个号码是400而不是800，显示出这家企业还是略微欠缺稍许迫力，如果是800全免费的接入号码我想用户更愿意去尝试的。这家公司刚处于起步状态，气势很大，只是前路漫漫，希望做好细节，在盈利和口碑上找到一个平衡点。当然继续扩大宣传也是很有必要的，也许这就是我现在这篇文章会出现的原因吧。

叽歪de - 你在做什么？

发表于 2007-09-20 更新于 2020-04-19 分类于 ADs ， IT Disqus：

不好意思我也来凑凑这热闹，说说[叽歪de](http://jiwai.de/?utm_source=feedsky&utm_medium= blog&utm_term=Microblog&utm_content=mobile_microblog)。 其官方评价如下：

“叽歪de - 这一刻，你在做什么？”诞生于2007年5月20日，北京。它是一个通过手机短信、聊天软件（QQ/MSN/GTalk/skype）和Web，进行组建好友社区并实时与朋友分享的微博客服务。创建者巧妙地把 .de的域名转化为“的”字的汉语拼音解，使每个用户在网站上的页面地址直观地呈现为“叽歪的某某”的汉语拼音。只要领会使用之道，你会发现它有趣而实用。叽歪de你，叽歪de我，通过用户的“手机微博客”组成了一个轻松而便捷的社区。

叽歪de这个咚咚咋一看很像饭否的，至于其前身始祖是什么我不是很了解，他在官方说明中说“创建者巧妙地把 .de的域名转化为“的”字的汉语拼音解”，其实我觉得jiwai.de这样的域名还不如GY.DE，这样既可以理解为“叽歪de”，还可以理解为“盖的”，正所谓“真不是盖的”嘛不过我想两个字母的域名注册起来实属不易，所以他现在这种解法也还行。 进入其官网注册的时候我发现他还要求提供真实域名，这个在维护社会和谐方面有点先见之明，实名制嘛，我想今后如果加入身份证就更完善了。整个注册过程比较流畅，登录后的设置内容也比较多，IM也比饭否多了个水木社区(虽然我没有去过)，还引入了OpenID，算是比较现代了。整个网页配色有点偏暖，也许是考虑到了天气即将转凉的缘故，不过据说这样的颜色容易让人视觉疲劳，如果可以自己改变皮肤就好了。功能略显单薄，在其“窗口帖”单元只做了Flash和JavaScript而缺少了图片及文字链接的选项，这无疑是对那些居无定所的人的一种轻视，要知道，正是这样的人，才是此种类别网站真正需要花时间气力去培养的用户，他们混迹於各大论坛，注册了各种奇奇怪怪的东西，如果他们把你的图片或者文字链接放在签名里，把你的网站作为他的主阵地，那更能够带来规模效应。不过总体来说，这个网站尚可，现在国内这个概念也还只是处于起步阶段，只要走好每一步，以用户为本，还是大有可为的。 试了试叽歪de’s Widget，效果差强人意。这里我不得不说的是，虽然叽歪de的Widget定制自由，但是有点太专业和麻烦，略显复杂，看来这个年轻的网站要走的路还有很远..且祝今后的路走好吧

击鼓传花

发表于 2007-09-18 更新于 2020-04-19 分类于 Finance Disqus：

小学时候每年寒假前的茶话会上，班主任都喜欢在没有节目的时候让同学们玩一种游戏：首先选取一名同学作为鼓手，然后准备一个标志物如假花，让被选中的同学蒙眼敲鼓，其他人在鼓声中把这假花依次传给下一个人，被选中的同学可以自行决定停止敲鼓的时机，这时传递假花的行为必须终止，花停留在谁的手中谁就必须表演一个节目，这种名字有一个很像成语的名字(小时候感觉四个字的词都像成语)——击鼓传花。 〇八Olympics选在了8月8号，对于中国人而言，他有个传统而俗气的寓意，然而他却正实实在在的体现着他所开张的日期所寄予的期望，中国股市、基市、期市都在迎来其前所未有的大发展，真可谓是核子聚变式的扩张。连很多连银行卡都不会用的人，都能随便使自己的资金猛涨数倍还心有不甘，连街边买菜的太婆们都在谈论今天什么什么基又涨了多少，整个资本市场越来越疯狂。 因为大家都知道奥运要来，然很多人知道大家都知道奥运要来，而有些人明白很多人知道大家都知道奥运要来，但有的人了解有些人明白很多人知道大家都知道奥运要来。 这让我想起了击鼓传花，奥运会就像那春节，老师是市场的创建和维护者，敲鼓的同学代表了政策，假花是炒作的载体，同学们集体扮演了资本的持有人。老师不参与整个具体过程，但是他可以决定由谁来敲鼓；敲鼓的同学需要蒙眼敲鼓，可是他不用塞住耳朵，也许他还可以通过弄松或弄薄蒙眼的红领巾的手段来使自己大致的看见假花传递的情况；同学们也不见得就必须以相同的时间和空间间隔以相同的时钟顺序来传递假花，可以把花抱在怀里在你认为鼓声即将停止的时候忽然塞给他旁边的同学(也许他会失败以至于假花一直在自己手里)，也可以把假花传回给传给你的人，也可以扔来扔去；假花有时候也有可能不止一朵，有的人可能会偷偷拿出备用的那朵。玩法多种多样，结局无非是两种，表演节目和不表演节目，不过老师和敲鼓者除外。 春节就要到了，很多人已经参加了这提前的茶话会，有的人在吃花生，把瓜子零食打包带回家的也大有人在，有的人已经表演过节目了，你想要表演节目么？喜欢吃花生么？或者，你根本没打算参加这场茶话会？ PS:发现一个很有意思的地方..
