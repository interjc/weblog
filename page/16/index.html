几个网络电视机顶盒的使用感受

发表于 2014-01-03 更新于 2020-04-19 分类于 digital ， Fun Disqus：

不得不说，自从移动设备逐渐普及以后，家里的PC机（台式机、笔记本电脑）已经有很长时间没有打开过了，特别是自从有了儿子以后，在家玩dota、wow已经成了奢望，所以PC机就更加没有用武之地了。在家上网使用移动设备已经成为了一种常态，并且，随着坑爹的广电一次次的挑战无耻的底线，我也停止了家里2台电视的有线电视业务的付费。 于是问题来了，在家也不能光上网啊，电视还是得看，用笔记本充当HTPC数月后，给人的体验并不好：

首先是遥控器的问题，理想状态下无线鼠标虽然可以取代遥控器，但是他挑战着使用者的视力，并且，不能悬空操作也是硬伤（鼠标下面必须垫东西），使用起来十分狼狈； 其次是内容问题，理想状态下只要有电脑就什么都能看的，但实际上并非如此，事实上PC视频客户端资源极其分散，而浏览器web播放功能非常弱不说，也不符合一般看电视的习惯，总而言之，非常蛋疼； 最后是效率的问题：用来充当HTPC的是一台预装着Windows Vista，只有2G内存的老爷机，开机速度极其慢，各个视频网站播放器的客户端又时常不老实，不会自己去退出后台进程，切换多个客户端观看在线节目以后，机器就会卡得不行。

写了这么多其实就想说，不使用一个有遥控器的专门设备来配合电视机是不行的。幸运的是，今年正好赶上了安卓机顶盒爆发的一年，先是有小米盒子2代发布，又有谷歌Chromecast以及一大堆跟风产品的跟进，然后乐视盒子和一大堆老单片机厂商的进入，最后天猫魔盒也来搅局，安卓机顶盒市场一片繁荣景象，然后还有一大堆忽悠小白的所谓智能电视（为什么说忽悠小白请见 王自如的科普），最后不得不提一下尊贵的 Apple TV，高富帅的选择。 然后我是怎么选择的呢，大致说一下经历：

首先选定的是小米盒子，因为刚开始不懂，就选了这个知名度对我而言相对较高的，官网肯定是抢不到的（时值2013年10月，不是个好时候）； 于是在某宝上找了一家看上去像黄牛的店，下单以后40天告诉我仓库进水，货全没了，需要再等多久不定时，一怒之下选择了退款； 正好赶上双11，买了一台海美迪Q2II（双核512内存廉价机型，199元）先顶着，因为家里已有几个月没正常看电视了，实在等不起，于是乎下单。拿到海美迪以后还是蛮兴奋的，毕竟是第一次使用网络电视机顶盒，装了N个APP也没见卡，果然还是专业设备做专业事，去除了通讯模块以后减负了很多吧可能； 后来在使用过程中认识了一些玩机顶盒的人，被推荐了乐视T1S（双核1G内存），算是上了贼船吧，当时也是双11前后399元做活动，送一年乐视VIP，于是乎就买了（现在真后悔，不如买C1S啊）； 前几天参加淘宝的活动抽奖，中了一台0.01元的天猫魔盒（双核1G内存）。

先简单说下海美迪这台机器吧，由于买他的时候就是想选择一款备胎，所以也没想着会有多好。不过拿到机器以后还是给我不小的惊喜，这款机器做工良好使用流畅，一般看电视足够用了，相比小米盒子只有一个MicroUSB接口，还需要再买一根OTG线才可以外接设备来讲，这款机器自带2个标准USB接口和1个SD插槽就略显厚道了啊（特别是对于我这种家里有一大堆闲置SD卡的人来说），系统没啥说的，就是一个安卓4.x（应该是4.1，没注意看，其实都差不多）外加一个自己定制的Metro风的Launcher。自带资源基本没人看，一般都看第三方APP里面的资源，所以这个launcher就不做介绍，有兴趣的可以自己去搜索（下面两款也一样）。只有一点不太方便的地方，那就是遥控器按关机键这个盒子他就真的关机了，然后再按开机就会冷启动，粗一想还觉得这挺好，但是误操作就坑爹了，会等半天（家里有小孩喜欢乱按遥控器的就更坑爹了），遥控器配7号电池这一点倒是挺不错（普通红外模式，遥控器遮挡有效）。 然后说乐视盒子，乐视T1S的硬件不说了，基本和新小米盒子一样，不同的是多了2个标准USB接口，另外由于使用了金属机身（是不是更有利于散热还要夏天才看得出来了），所以多了一个外置天线。遥控器使用了红外模式的纽扣式电池，奇葩的是没有音量键着实蛋疼，不过也就凑合用了。乐视盒子的固件开发应该是所有电视盒里面最差的了，跟iCNTV合作的自带内容暂且无视，副作用是开机后主界面上巨大的广告图片竟然不可点击，自带Letv TV版APP每次打开都需要等待5秒以上（iCNTV 播控），乐视一年VIP我看了一下，视频质量倒是不错，就是老旧了一些，看了一下乐视的官方论坛，发现固件更新很慢，BUG奇多无比（升级了号称最稳定的V039固件以后也是问题多多，内置应用市场闪退、应用列表界面显示错位等问题时有出现）。不过乐视盒子也不是一无是处，至少操作灵敏度还是不错的，机器性能感官上比之前的海美迪还是要强一点，本地播放1080P视频也基本没有什么压力，满足了我的主要需求。另外我觉得最最实用的一点是可以充当 AP 使用，由于我家无线路由是绑定了 MAC 地址白名单的，所以每次家里来人来客，想给他们开 WIFI 就显得有些不太方便了（需要经历一次PC老爷机的开机过程），现在直接在遥控器里面进乐视盒子的设置项开启 AP 功能即可生成热点了。就因为这个，乐视盒子得以保住了留在客厅里面使用的地位。最后还是要吐槽一下乐视盒子，开机竟然有广告，广告竟然有声音，还奇大无比，由于遥控器没有音量调节按钮，所以广告声音自然是关闭不了的。最后只有 ROOT 去广告了事了（相关破解资源）。 接下来是天猫魔盒（0.01元，双核1G内存），天猫应该是想尽快的抢占客厅市场（天猫魔盒内置了聚划算、天猫等应用，基本等于一个大广告了），所以天猫魔盒采取了以各种理由白送的模式。忙活了好几天，幸运的弄到了一台，拿回家以后发现机器做工很不错啊，自带一个标准USB接口和一个MicroSD卡插槽，也算是厚道了，开机以后的天猫LOGO虽然是广告但感觉还是挺漂亮的，不得不赞一下阿里的设计团队，系统方面没啥硬伤，自带资源基本也是无视的，值得一说的是，天猫魔盒是唯一一款不需要ROOT就可以自由卸载大部分自带应用的盒子，为这种气度和胸怀点个赞吧还是。天猫魔盒也使用了纽扣电池，但似乎并不是红外的方式，因为就算有遮挡物也是一样可以使用遥控的，这一点在卧室里面使用的时候就显得比较实用了（可以隔着被窝遥控啊，有木有）。 经历说了一大堆，最后来一个总结吧：天猫盒子无疑是我接触的这几款盒子里面性价比最高的（就算你不是白送获取，有很多抢到了盒子的朋友在某宝上也在以150-180元的价格在出售，这个价格买这个配置的网络机顶盒也是不吃亏的），乐视盒子的任何版本都不建议购买了（主要是固件不给力，开发实力太差啊有木有），海美迪这个公司还不错，可以考虑他们家的产品。 利益相关：普通用户，天猫（相当于）白送我一台盒子。

安卓老爷机上管家类软件的选择

发表于 2013-12-13 更新于 2020-04-19 分类于 IT Disqus：

写在前面的话：那些准备说手机不需要安全管家、可以裸奔，安卓手机可以无视内存管理的人趁早滚蛋，一看就是没用过 512MB 内存安卓手机的人。另外，别跟我提“绿色守护”，2.3.x 用不了。 言归正传，就从我这部老爷机（Defy+）的使用过程说起吧： 刚买这部机的时候，512内存+姜饼系统还算是主流，所以安全管家的意义还仅仅停留在拦截垃圾电话/短信、清理通知栏广告等功能上，对性能优化的要求还不是那么迫切。本着帐号复用以及功能多样化的需求，选择了腾讯手机管家，因为他拥有很多功能可以让我删除手机上其他不必要的APP，比如截图（没错，defy+没有原生截图功能）和同步通讯录等，于是就选择了他。 时过境迁，手机上各种主流 APP 的版本迭代速度是很快的，于是这台机器不可逆转的沦为了老爷机的行列，我不是一个喜新厌旧的人（也可以说是抠门吧），一般遇到这种情况我的第一想法一定是内部挖潜（也就是内存优化），当我发现腾讯手机管家的内存管理和自启动管理功能几乎是形同虚设以后，就毫不犹豫的弃用了他。 在一些评测文章中进行了比对以后，我选择了 LBE。这是我用的时间最久的一个手机管理软件，我发现他真的可以让我不想要的应用不开机启动，当我选择 kill 掉一个进程以后，它不光会真的被关闭，而且会真的没法后台自启，另外他的快捷工具栏也很有效率，让我可以删掉手机上的海卓和从桌面上去除一系列的开关类 widget 了。但是好景不长，在天朝，一个好的产品被改成一坨屎的情况是时有发生的，特别是 LBE 升级到 5.0 测试版（添加了所谓的免 ROOT 功能以后），BUG 丛生，好用的内存管理功能荡然无存（看上去杀掉了进程，过会儿再看其实没有杀掉，开机自启动也是同理），更令人无法忍受的是，快捷开关功能会造成系统本身的重要功能无法正常使用（比如有时候会造成返回键失效，或者直接黑屏，软件里的相关设置项设置失效会不停的被重置），在一次疯狂的 BUG 中，我甚至无法从快捷工具界面退出到桌面（按什么都不行），最后只有进入 LBE 软件卸载了它自己，才算完成了这一段 LBE 使用经历。 于是又进入了新一轮的手机安全类软件选择的困扰之中，万般无奈之下选择了 360手机管家。这实在是一个无奈之举，因为用户使用量靠前的管家类 APP 就那么几个，我又极度不喜欢金山，于是就它了。360手机管家的整个使用过程只有用差强人意来形容，用过以后我才知道为什么有人不喜欢 360，它确实是设置了很多陷阱来误导小白用户安装它的软件（比如 360 浏览器和 360 手机助手），但对于我来说这倒是没什么，只要我不去点，它也还没有无耻到自动替我安装的程度。360 删除垃圾软件的能力是我用到这个时间段最好的，他将我可怜的手机内置容量提高了2、3倍，这是我以前用 LBE 的时候做不到的。不过问题也是存在的，最致命的还是无法真正的阻止应用后台自启，于是我只有一遍又一遍的去点击清理内存的按钮，不然手机就会卡得要命，另外，他的快捷工具栏也不太快捷，不能通过设置快捷键来唤出，只有在桌面添加一个悬浮球（这是有强迫症的我感到很难受的地方）。并且真正的问题在于，360 的内存管理逻辑让我很不习惯，当我添加某些 app 到一键清理白名单之后，还是希望能在快捷工具窗口里面可以选择性的手动关闭它（比如QQ、微信、音乐一类的可能会经常运行在后台的应用），就像 LBE 的逻辑那样，但是 360 的做法是只要你把它添加到了白名单你就没法关闭它。再者，360 的去除广告功能也不太彻底，于是没多久我就毙掉了 360 。 当我看到街头巷尾铺天盖地的腾讯手机管家广告的时候，想着以腾讯的财力，把一个东西做好应该只是时间问题，所以准备再给他一个机会。当我重新安装了它以后，发现确实比以前好多了，但改进也仅仅只是停留在了 UI 上，深度使用以后我发现他有着和 360 一样的毛病，内存管理的逻辑让人很不舒服，另外他也没法真正的阻止应用后台自启。看着常常卡到没法正常操作的手机，我万念俱灰，有了换手机的念头，可是当我去手机市场转了一圈回来，却发现没有一款看得上眼的手机了（我是移动用户，别跟我提换号，我不换号），没有 TD 制式的手机实在不想买，因为无法忍受换了一个手机以后还要使用龟速的 gprs 的痛苦，但是支持 TD 的除了我最恶心的三爽以外，就只有小米3和魅族看得上眼了，可小米3着实不好买，另外做工也很差，魅族手机的 smartbar 又过于奇葩了，所以想想还是算了，再等几个月吧。 生活总是要继续的，我又换回了 LBE，几个月的迭代以后，当时遇到的 BUG 被修正了，内存管理也恢复了我一开始使用它的程度。久违的流畅使用手机的体验回来了。 希望我以上的经历对你选择一款安卓老爷机上的管家类软件有所帮助。 Have fun!

我使用的 Chrome 扩展

发表于 2013-09-13 更新于 2020-04-19 分类于 Tools ， Web Disqus：

从第一版 Chrome 发布即开始使用，至今有多少年头早就不记得了，Chrome 有一个好处，配置（包含扩展）可以同步到服务器端，于是同步下来一些好东西，现在列举出来分享给大家，也算是一个资源的保存，以备将来查询。 扩展有点多，所以我将它们分为了三大块：推荐、私人和备用。扩展的列举顺序按照推荐程度和重要性排列。

推荐

Proxy SwitchySharp 科学上网的必备工具，配上那啥（不能说的秘密），上网妥妥的。妈妈再也不用担心我404了～ FaWave(发微) 目前就用他来转发好玩的东西，虽然时常出现 BUG 但依然无法掩盖他的伟大，暂时还没找到替代品 Remove Google Redirects Google 搜索结果有时候会添加重定向链接，这是我们不愿意见到的，有了它，链接都干净了。该扩展以前介绍过，此处就不再赘述了。 Adblock Plus 看名字就知道了，屏蔽广告用的，他的好处是可以使用起来比较傻瓜，安装就运行了，也不用配置什么，还可以添加白名单（白名单的好处是优酷有时候会封杀广告拦截软件，此时将优酷加入白名单就好了）。 图片搜索 神器啊，我还能说什么呢，以图搜图的时候特方便，不需要专门去先打开一个图片搜索页面，谁用谁知道。 惠惠购物助手 比价软件，网易做的，还算比较用心的产品吧，虽然有时候跟某些网站的界面有冲突，但也无伤大雅了。 ActiveX for Chrome 网银助手 啥也不说了，该死的国内银行。该扩展有时候有效有时候无效，凑合用吧。 Pocket (formerly Read It Later) 保存长文章同步到手机上看的时候有点儿用。 印象笔记·剪藏 同 pocket，不过 evernote 的功能似乎更强大一点，我是两个配合使用，分别用来存不同的内容。

私人

Google Mail Checker 自从弃用 GTalk 以后，就指望他提醒邮件了 JS Runtime Inspector JS 调试工具 Chrome Sniffer 查看该网站用了哪些技术，蛋疼的工具 查看背景图片 如题 Eye Dropper 一个取色的工具 Classic Retweet 刷推的时候，提供使用传统 RT 方式转发的快捷按钮 goo.gl URL Shortener 一个短网址工具，Google 出品，由于实际上已经被墙了，所以只是内部交流的时候用用。它的好处是永远都不会被和谐，因为已经被和谐了..

备用

网页截图 - Webpage Screenshot 有时候没有打开 QQ，又不想打开臃肿的专业截图软件，却要进行截图的时候，它就派上用场了 Enable Copy 有些网站屏蔽了右键，它就派上用场了 ZenMate for Google Chrome™ 有一个科学上网的工具，但是我已经有 Proxy SwitchySharp 了，所以他只是个备胎啊 :) 如意淘 阿里系的比价软件，由于跟其他电商的竞争关系经常遭到封杀，数据更新很不及时，所以弃用了，但依然不失为一个不错的备胎
