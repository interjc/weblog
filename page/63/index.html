新年第一贴:对Google的展望

发表于 2007-01-01 更新于 2020-04-19 分类于 IT ， Noise ， Web Disqus：

首先祝大家新年快乐！Google如期推出了新年LOGO，挺好看的，偶然发现Google.cn下的Google Labs已经归属于.cn了，名称也更改为了Google实验室，我一般直接用Google.com，所有也不知道这些改动是什么时候进行的，不过也算是有所进步吧。”正常”情况下，网页快照依然无法使用,看来李开复们要走的路还有很长。 我不是Google粉丝，虽然我使用很多Google的服务，但是那仅仅只是因为他更好而已，如果有其他的好东西比他杰出，我会毫不犹豫的更换；事实上，我还使用很多Yahoo!的服务，可我对Yahoo!却是难得有什么好感，最可恶的是他的爬虫天天在我blog上面来绕，却只收录了惨烈的一条，而baidu偶尔来玩一次，却已经收录了10000条(虽然断网时期被他砍掉了2/3).. 话题不扯远，现在就来说说新年对于Google的几点想法： 在技术进步的层面上好多人已经说了，我这里也就不再赘述，我主要说说市场层面上的。 1.开放中国市场的Gmail注册，这一点很重要，最好还取个名字叫做谷歌邮箱什么的，Google的一系列产品以此为本，也许开放注册在中国会造成ID的滥用，可是总比没什么人用好吧，况且现在的邀请制本身就存在小范围的滥用了，我想以Google的财力这不算什么大问题。有了自由注册的Gmail，才会有更多的人投入到GTalk、Picasaweb、GReader等等产品的使用中来； 2.GTalk中文版需支持文件传输功能，但最好不要开发群功能(而用Groups、Gmail取而代之，群功能的吵闹恐怕是高端用户无法忍受的)； 3.Google Earth一定要推出简体中文版，据我实际观察，在所有的Google产品中，最能吸引用户的就是他了，要是能和Google本地结合起来就更妙了； 4.Google收录勤快一点，不要个把月了还不更新已经被更改URL的搜索结果； 5.最后一条，也是重中之重，没了这一条说什么也是白说，那就是，搞好和政府、人民的关系，多修一点希望小学，多做一点领导工作，最好能把GFW的矛头转向隔壁去，那你就成功一大半了。 就说这么几条吧，说多了一年做不完，还请Google努力。 PS：今天有位卖菜的达人又写了一篇骇世奇文，叹为观止。希望他的blog暂时还没有被GFW。

2006年度汉字:网

发表于 2006-12-30 更新于 2020-04-19 分类于 IT ， Noise ， 文化 Disqus：

日本汉字能力鉴定协会公布，“命”字当选为日本2006年的年度汉字。这个毕竟是日本的推选结果，据说主要是因为日本王室今年迎来天皇长孙，以及日本国内发生了一系列学生受辱自杀事件和虐待儿童事件。那么中国呢？我认为是”网”。 网络存在久矣，只是真正在中国推起热潮的还是2006年，怎样才算是热潮呢？打个比方，以前陈天桥的理论是，到处都在卖传奇点卡，有个老太太他不知道传奇为何物，也不会上网，但是他发现大家都在买传奇点卡，于是她也跑去买了一张，这种状况，就叫做热潮。 如果说去年的大妈只是处于别人坐着她站着别人吃着她看着的状态的话，那么今年，她绝对是已经深入的参与了进来，并且和大家打成了一片了，有可能还成了群众议论的焦点或者做着这样的梦并努力着。今年网络的一大特色是平时仅仅是聊聊QQ玩玩网游的同学们都参与到如火如荼的博客、播客、掘客、极客、访客@#$%&``中来了，门户和大型网站煽风点火，大家都来为网络提供内容，都成了信息制造者而非看客..时代周刊年度人物是YOU，也就是显示器前按动键盘的你，而这个年度人物活跃的平台正是”网”。YOU们把自己的生活都搬到了网上来，对于社会事件反应速度加快，人们可用来参考统计的数据也多了起来，于是乎另几个关键词”GFW”、”恶搞”、”广电总急”、”实名制”浮出水面，而这一切都借着”网”的深入人心而活跃于大众;央视对于体育赛事的漠不关心成就了网上直播的繁荣，民众生活中的压抑成就了”恶搞”，伴随超女而来的草根文化使很多人有了一夜成名的梦想，而”网”就是最好的平台，于是造就了来者皆是”客”的繁荣。 年底，到了谢幕的时候，”网”又有拉票之举，借由台湾地震之势，中美光缆断开了连接，”网”以这样的独特方式吸引着人们的眼球，他要告诉大家，现在的”网”已非数年前那次中美光缆断裂时的青涩小”网”可比了，如果几年前那次断线带来的是shake的话，那么这次绝对是hurt了，无数依托于”网”而存在的公司、个人因此蒙受的巨大损失，已然超过了网络本身。 没有了”网”，你的2006(2007)年是什么样子？ PS:写这篇文章的时候我依然因为某些原因无法直接连接这个身在美国的服务器，现在的网络半残状态已经让人很不习惯，无法想象完全没有网络的生活。偶然发现feedsky可以抓取我的blog了，kreep说抓瞎也可以抓取feedburner了，这是复苏的迹象么？

圈内人民一切安好

发表于 2006-12-29 更新于 2020-04-19 分类于 IT ， Noise Disqus：

和世界断开畅通的联系还只有几天而已，可是却感觉已经过去了个把月，对于我们这些喜欢鼓搞国外的东西的人来说，是为灾难，也许你可以通过代理或者其他途径online，不过你的客户、你的朋友、你的圈内(暂且把断在网内的部分称为圈内吧)读者们却也许不知道，一次地震的危害不仅仅是物理上的，更大的是软损失，很多跨国经商的企业无法运转，国内Officer联络的重要途径MSN大面积无法正常登录，这样的损失不是表面上可以估算的。 不过我们还有Google，这样的天灾之下，Google巨大的投入带来的优势展现无疑，我们依然可以每天挂在GTalk上和朋友们保持联系，GMail使我们的邮件畅通无阻，可以通过Google Reader查看已经无法访问的Feedburner上的信息，Google上的搜索似乎比以前更加顺畅了，非常时期，GFW也良心发现，似乎这几天出没的不多。 这几天都在不停的下载BT和看电影的过程之中，也许断网之后，娱乐才是我们的最后归宿，我打开了许久不曾动过的QQ，上面的人们丝毫没有感受到恐慌，很好；我说我断网了，他们说你投诉武汉电信吧，我家挺好的，我无语；也许，这次断网的影响真的没有我们想象的那么大吧，毕竟不是每个人都喜欢跟世界一起呼吸；圈内人民一切安好，只是上网不知道干嘛了 :) PS:百无聊赖之际看了下水道的美人鱼(在线胆小勿点、刚吃过饭或者准备吃饭的勿点)..虽属N年前的老片子，可是依然可以带来视觉上的震撼.. 其实天灾怎么也是比不上人祸的.. 这次是天灾，其实还好..

体验国内大局域网的美妙生活

发表于 2006-12-28 更新于 2020-04-19 分类于 IT ， Noise Disqus：

海缆断了，于是我自己的Blog不能访问，Skype无法登录，还好不用MSN、没在网上做生意，不然那就真的玩完了。幸运的是，Google可以访问，GMail和GTalk使我依然可以与外界保持联系，Google Reader使我依然可以阅读以前可以读到的文章，只是好文章没法del.ious.us了。

使用国内服务的朋友们有福了，这些天看你们玩的热火朝天还是挺羡慕的。我就没有那么幸运了，细数下来，我使用的网络服务基本上都是国外的，现在都弄不了了，每天除了挂GTalk根本不知道上网来干什么的。百无聊赖之际可能最好玩的就是BT上下载一些电影来看看了(这个等网好了再介绍吧)。

我们的网络如此脆弱，倒不是说他这个抢修如何如何，有这么多钱，怎么只修了这么一条线路呢，此为天灾无法避免，但如果遇到人祸怎么办？现在属于和平共处损失的只是些微经济，那么如果战时出现这种状况呢？说多无益，只是希望这些问题能得到改善吧。

写这些估计不用Google Reader或Gmail订阅我的朋友一时半会儿是看不到了，姑且留作纪念吧。
