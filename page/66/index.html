PayPerPost:又一Blog Review赚钱方法

发表于 2006-12-18 更新于 2020-04-19 分类于 IT Disqus：

之前有个ReviewMe，只要你按照要求写了帖子，就会收到Check，现在又看到一个类似的网站–PayPerPost。不同于ReviewMe的先看此Blog是佛否合要求、而后发出邀请、最后即使这个Blogger写了一篇仅仅只是提到了相关信息的废话文章都可以得到钱的高门槛低效率，PayPerPost采用的是低门槛、先写后审核的模式，每个人都有机会来赚钱，只要你写的好，这样保证了广告商的利益，也给了所有人自由参与的权力，不过这样的弊端也显而易见，也就是什么样的文章会通过而什么样的又不会通过的问题，当然，你可以自己查询，没有通过的文章大可以删掉，呵呵。并且，你也可以作为一个广告商的身份参与进去，要求别人给你写贴。这个网站更像一个软文自由交易市场。 通过PayPerPost简单的手续，首先注册，然后添加你的Blog，接着在List里面寻找你感兴趣的话题，点击进去接单，然后写贴子，最后在Post里面查看是否通过，然后就在家等着支票的飞临了。 这种模式是很不错的，可是我找了半天了没有找到适合我来写的话题，也许英文世界的Blogger会从中获益更多吧.. :( 中国Blog圈里什么时候才会出现这样的服务呢？ 最后，PayPerPost的网站设计也比ReviewMe好看多了.ads on blogs

关于Google的企业化服务

发表于 2006-12-17 更新于 2020-04-19 分类于 IT ， Web Disqus：

亲戚开的公司需要一个简易的网站来存放他的公司图片和联系信息，于是找到了我，这是一家传统企业，主营与IT毫不相关的贸易，他们的业务都是通过电话和面谈来完成，需要网站的理由是他们认为每家公司都应该有一个自己的网站。基于这样的要求我马上想到了前几天自己刚刚开通却没想到用来干什么的Google Apps提供的服务，域名+Google Apps绝对是成本最为低廉却很有效的对企业的宣传方式。 这个工作完成起来很简单，对于充斥着电脑盲的传统行业而言，记住一个.com以外的后缀是很难的，所以首先买了一个以这家公司的汉语拼音为前缀的.com域名，紧接着申请了Google Apps，而后将其域名绑定至Google Pages，邮箱使用Gmail的托管，这样一个传统行业的企业网站雏形就完成了。Google Pages本身的功能的确有限，对于js的过滤使很多应用无法施展，不过这个可以利用上传制作好的html文件来弥补；托管于Gmail的企业邮箱ID可以使用在Gtalk上，这一点很好，可以使员工不用记忆ID和密码即可方便的收发邮件和交流；运用Picasaweb可以方便的批量上传和管理企业的产品图片。这些做完以后，这家公司的要求基本上就完成了，有了以自己公司名字为标志的主页、邮箱，公司的电话、地址和图片都可以置于网上推广了，更重要的是，除了那越来越廉价的域名以外(这个并不是Google的问题)，其他一切的一切都是免费的。对于这家公司而言，唯一的遗憾就是Picasaweb的public页面无法与其域名进行绑定了，这样就会暴露自己所使用的服务，也算是个不大不小的瑕疵了。 以上所说的仅仅是以这种对网络功能要求不多的企业用户，虽说功能刚刚够用，却不能说齐备。我想，Google为什么不能把旗下的Google CheckOut、Base、Groups、Blogger、Picasaweb、Docs全部整合到Google Apps里面呢？这些从技术层面上对他们来讲并不是什么难题，而这些资源本来就是免费开放注册的，只是企业用户当然不希望自己的企业blog是以blogspot.com结尾的。试想，当一家小公司在只花了几十元钱购买了一个域名以后，就可以免费拥有自己的主页、图库、产品数据库、结算系统、论坛、邮局、企业及员工Blog、免费的在线文档处理(更方便于合作编辑和传阅)，那将是一个性价比多么高的小型传统企业解决方案，如果在企业内部使用Gtalk时可以使用Gtalk群功能的话就更妙了，其实这个功能实现起来并不复杂，更无需修改软件，仅仅是借用MSN群的办法，大家共同添加一个作为server的ID。那么，和前面所说的设想结合起来看，这家公司就拥有了极为完善的一套网络资源，而价格是低廉的不能再低廉。 对于传统中小企业而言，纯粹为了运营自己的网络系统就耗费大量人力对他们来说是不化算的，而如果上文成真的话，那么这家企业要做的仅仅是聘请一名兼职技术支持就可以了。 Google Apps还处于Beta阶段，一切皆有可能，希望我说的这些不会仅仅只是成为设想而已。 PS：我在自己blog上的Google广告中发现了一个有趣的现象:)

发表于 2006-12-16 更新于 2020-04-19 分类于 IT ， Tools ， Web Disqus：

今天又冒出来一个搜索引擎–有道，看名字应该是盗(?!)亦有道这个意思，一开始认为这是网易的跟风作品，应该不会很好，可是去看了看以后发现还有点意思。网站刚刚做起来，收录量还不是很足，不过看上去比百度博客搜索的情况好一点，不会只护着自己的孩子，基本上大型BSP的数据都比较全，搜搜看如果你的blog没有被收录可以试试这两个登录入口[入口–网页搜索][入口–博客搜索]。(UPDATE:提交后收录速度挺快的，今天我就看到我的主blog已经被收录了 :) ) 搜索网页的时候会逐字提示最热门的关键词，这个设计很贴心： 把鼠标移到博客搜索结果中的)小图标上，会”弹”出一个小窗口显示帖子内容(如果结果是一个blog的话会输出被收录的最新贴中的内容)，并且他还会尽可能的收集到blogger的头像： 搜索结果可以订阅，而且已经为你预设好了目前流行的RSS阅读器(抓瞎、Google、Bloglines)： 除了搜索以外这个搜索引擎还提供另外一些服务，比如在博客搜索搜到一个blog后点击”博客档案”，会出现一个相关度还算可以的评价，并统计出了你的帖子篇幅和发帖时间分布： 你还可以在其”海量词库“中搜索单词释义；可以在博客男女中看看你的文风有没有娘娘腔，测了测我自己的，他是这么说的：
