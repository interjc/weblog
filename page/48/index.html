新数码混搭主义

发表于 2008-04-21 更新于 2020-04-19 分类于 ADs Disqus：

引领“新数码混搭主义”风潮，HP Pavilion dv3000系列笔记本电脑为你创意生活万千精彩！它融汇13.3英寸高亮宽屏、高性能显卡、生物指纹识别技术、高感光摄像头、3D DriveGuard移动保护等领先技术于一身，自成格局，堪称混搭集大成者。无论你怎样搭配生活，它都能与你心心相通，尽情风尚每一面。 HP可是我喜欢的牌子，如果我下一台电脑不是Mac的话，那一定会是一台HP笔记本，这绝对不是奉承。 HP dv3000集近乎完美的外观和强劲的性能于一身，Intel新一代45纳米Penryn 新处理器性能更强劲、功耗更低，发热更小；机身重量为2Kg（含4芯电池），便携性佳；HDMI接口，为高清输出提供了极大的方便；奥特蓝星扬声器和独有的QuickPlay，能提供非凡的娱乐体验；130万像素高感光摄像头、8400GS独立显卡（256M显存）、内置麦克风、1394接口，指纹识别、蓝牙等高端家用笔记本有的东西它都俱备。处理器主频（GHz）：2.1 迅驰技术：迅驰四代 处理器编号：Intel Core 2 Duo T8100 标准内存容量（MB）：2048 硬盘容量（GB）：160 显示屏尺寸（英寸）：13.3 显示芯片：NVIDIA GeForce 8400M GS 重量（kg）：1.98。这个配置对于笔记本来说也算是够用了，价格也不算是高得离谱，对于HP这个品牌来说也算是可以接受的。 我们的生活越来越丰富多彩，当我们的生活也越来越忙碌，当我们用同样的时间完成更多的事情，用不同的装扮搭配使自己更引人注目，“混搭”也成了我们生活中的以部分。像HP一样，当“混搭”无处不在的时候，你会发现原来枯燥的生活可以变得如此美好。 其实所谓混搭就是把两件或多件事物混合起来合成一种新的事物的行为，它是一种生活方式，更是一种态度，就像你用电脑可以同时干很多事情(听歌、下载、写blog、看网页)，而在用电脑的同时你可能还正为自己煮着拿铁，这就是混搭。而数码混搭就是说把你的所有数码产品通过你的电脑这个平台(当然这台电脑也是一个数码产品)联合起来更好的为你精彩的人生服务。 由惠普（中国）有限公司和优酷共同举办的“我的数码混搭生活视频大赛”现在已经火热的展开，征集符合DV3000系列笔记本电脑的最佳视频。各路网友纷纷参与，上传视频作品来表现出数码“混搭”的生活理念，最优秀的作者还将有机会获得引爆数码混搭风尚的DV3000笔记本一台！ 这次大赛基本上没有什么特殊的要求，只要你注册了优酷账户，自己制作一段或多段你认为最能够诠释数码混搭这一主题的视频并上传，打上”HP混搭”的标签，你就已经参赛了。 其中前十名(大奖2名，二奖8名)将分别获得HP Pavilion dv3000笔记本电脑和IPOD NANO3。即使你没有上传任何视频作品，你也可以通过转发此次大赛相关作品来参与，转发的前15名可获看客“奋斗”奖，奖品是豪华木质笔记本电脑桌。此次活动起于今年4月10日，将于5月30日揭晓最终结果暨活动结束。 节目最后我也转发俩，做一把看客。 PS：实在不好意思，又来一篇话题作文，真可谓不在沉默中死去就在沉默中爆发，于是乎，今天feedsky连续给我两篇话题作文，是不是他们也知道我的域名和空间该续费了呢？好人那，谢谢！

攀珠峰 传圣火

发表于 2008-04-21 更新于 2020-04-19 分类于 ADs Disqus：

北京奥运将至，2008年5月，奥林匹克的火炬将在珠穆朗玛峰点燃，它燃在世界的最高点。火炬燃烧着我们心中火热的激情，牵动着我们的心。4月19日，华硕的珠峰志愿者也已经启程，向着神圣的珠穆朗玛峰进发。华硕珠峰志愿者是幸运的，能在世界之巅亲自见证奥林匹克火炬的传递，他们也是勇敢的，他们在挑战身体的极限、未知的困难放下手中的工作倾情以赴，去迎接奥运火炬祝福奥林匹克，他们在 “挑战人生的新高度”。 登珠峰是一个很酷的活动，可是要亲身参与进去，你的工作性质、财力和身体条件必须符合条件，我们大多数人即使向往也只能远观而不可亵玩焉了，虽然没有机会成为珠峰志愿者，但我们依然可以各种不同的方式，向攀登珠穆朗玛峰的志愿者们送出我们的祝福，从身边的点滴小事做起，为即将到来的奥林匹克聚会，作出我们自己的贡献！ 作为一个blogger，你可以参加华硕征文大赛，这样你虽然不能亲临现场，但依然可以通过你的智慧参与进来，还有机会获得华硕提供的超便携式电脑易PC（EeePC）。 华硕这家IT企业，其产品的高端用户群是那些高级IT人士及事业有成乐于承担社会责任的人，所以这次活动由他们来牵头，比较合适。华硕的品牌形象和这次活动也比较相合。我们blogger都是草根大众，参与这次活动还有机会可以获得传说中简单实用的EeePC，也还是很不错的。 这是Feedsky在把我的blog定价升至60后首次给我发任务，而这次活动的文章据我所知在去年就已经开始了，如今此活动还在进行，足见华硕对此次活动是费了一番心血的。据说此次登珠峰的大本营所在地已经是5200m的高地了，我们的奥运圣火还将通过这支国家登山队到达世界最顶端，这真是无比的荣誉，而且恐怕也是前无古人后无来者的壮举了，在心说“lucky guys”的同时我也衷心的祝福这次活动可以顺利进行。 今年中国诸多不顺，希望这次登山、这次奥运都可以顺利而圆满的进行，让我们转运吧，加油了！

奋斗

发表于 2008-04-15 更新于 2020-04-19 分类于 Fun Disqus：

前段时间看了个电视剧，国产的，有点小意思，也许很多人都看过了，也许还有很多人没看。反正我有点话想说。 这个片子叫奋斗(土豆、VeryCD、BT)，讲述的是一帮80后大学毕业以后遇到的事情。让我明白了很多道理： 1.故事的一开头就让我们少了一个可以值得期待的主角，那个跳楼的我还真想看看他毕业以后怎么混，他没钱没关系没学历，如果混出来了可以作为很多人的榜样，可导演也不知道这样的人如何才能混出来，所以在剧本里把他谋杀了； 2.故事紧接下来我就看出这是一个在家吃软饭、工作靠父母、谋生靠朋友的故事，剧中层出不穷的有钱爸爸向我们说明，还是站在巨人肩膀上看得比较爽； 3.故事的主旨仿佛是在说，老一辈都是好样的，在文革中存活下来，在改革开放的中国特色社会主义建设大潮中发财，可80后净是些倒霉儿子傻女儿，乱花钱已经没什么大不了了，让他们接掌自己事业可就败家了，典型人物：陆涛； 4.大学生毕业以后如果没有有钱爸爸，做自己本专业的事情估计是很难发财的，典型人物：陆涛在Cooper的那位学长； 5.自己做生意是个不错的生财之道，不过要找对合伙人和老婆，典型人物：华子； 6.大学生毕业以后大多数是做销售，刚开始也许嫩点儿，可做出点业绩来以后就不同了，当然你得适当有点人脉，典型人物：向南； 7.市侩的女人适合过日子，自强的女人自己去强吧，典型人物：杨晓芸、夏琳； 8.电视剧里做广告效果比放电视剧时插播广告有效的多，电视剧里的广告可以从主角的嘴里“不经意的”冒出来，典型产品：A1牛肉、奥迪A4、点八中南海…… 9.房地产很赚钱，典型人物：倾家荡产也要改行的米莱的爸爸； 10.女人和女人之间永远不可能有真正的友谊，男人无法理解，典型人物：勾心斗角的杨晓芸、夏琳们； 11.找乡下女孩谈朋友要小心，注意观察她是喜欢人还是喜欢户口，典型人物：露露； 12.人善被人欺，典型人物：米莱； 13.龙生龙凤生凤永远只是奉承的话语，典型人物：陆涛和他俩倒霉爸爸； 14.如果演员演技不好，常常记不住台词，脸部表情僵硬，说话像背书，可以多拍摄他的背部和侧面，典型人物：马伊俐； 15.这个片子不该叫《奋斗》，我觉得叫做《80后的败家仔》比较贴切一点，80后的奋斗不是这个样子的，至少我们大多数人没有这样的有钱爸爸和有钱讲义气的同学； 16.国产电视剧又有新题材可以说了，一个《士兵突击》养活了多少部军事题材啊，这次看《奋斗》的了； 17.男人有点小钱都爱买SUV，没钱只好买奥拓，都想开奥迪； 18.未完待续.. 以上条目没有先来后到的关系，基本上是想到一条写一条。 电视剧是上个月看完的，好多情节现在也忘了，现在再不写过几天全忘了，大家凑合看吧。

换名字

发表于 2008-04-12 更新于 2020-04-19 分类于 Blogging ， 琐事 Disqus：

本Blog自公元2008年4月12日起正式更名为 糖醋排骨 这个blog的名字起先的几年里一直都叫做Helled Heaven，helled是自己造的词(至于它本身还有意思我之前是不知道的，自认为是自造的)，blog名字的意思是变成地狱的天堂，由此可见本人以前是一愤青，可是由于国家形势一片大好，而且又那么和谐，所以我在blog上也不那么愤青了，这个另很多人费解的blog名称也显得那么不合时宜。 再加上这个blog比较个人了现在，所以名字我也想换个个人一点的，喜欢吃糖醋排骨，所以名字也改成这个吧。 由于最近很忙也很懒，本来有一揽子计划来对这个blog进行一番大手术，没想到只是换了个小马甲，实在是不好意思。什么时候有空再来进行吧。 希望我大家继续支持我，3Q~
