京东无法登录的曲线解决办法

发表于 2012-07-17 更新于 2020-04-19 分类于 Web Disqus：

京东是我最常用的购物网站了，不过他的网站做得确实有点坑爹。就说这登录吧，我用 Chrome 浏览器登录，注销以后，就怎么也不能通过登录按钮（登录地址）去登录了。清空缓存 Cookies 什么的能做的都做了，就是不行，但是其他浏览器又可以。我总不能因为一个网站的 BUG 抛弃一个浏览器吧.. 真坑爹.. 无比蛋疼之际，各种尝试之后，被我发现了这个无厘头的登录办法： 1. 进入京东首页，随便点击一个商品链接； 2. 点击 加入购物车； 3. 点击 去购物并结算； 4. 点击 去结算，这时候就会弹出一个小窗口叫你登录了； 5. 在小窗口里面登录，建议勾选自动登录； 6. 登录成功了，随便点击的这商品，自然也不会去付账，先无视他，直接关闭窗口或者回首页即可； 7. 从购物车里将刚才那件随机商品删除即可 后来我又发现了一个更简单的办法，在未登录的情况下，点击右上方的 我的京东 链接，在出现的登录框中即可正常登录了..

QQ 旋风霸占 443 端口问题

发表于 2012-06-25 更新于 2020-04-19 分类于 Tools Disqus：

在 Windows 上既使用 QQ 旋风又使用 XAMPP 的朋友，可能也遇见过 QQ 旋风打开之后 Apache 无法启动的情况。 搜索了一下，原来是霸道的 QQ 旋风把 443 端口霸占了（via），之前各种下载软件霸占 80 端口这问题倒是好解决，把软件里面的 BT 端口换一个就成了，可是这个 443 端口，貌似是固定的，反正在 QQ 旋风里面是改不了，而为了离线下载又不能换用别的下载软件（屌丝是没钱冲会员的），如此一来只有修改 Apache 了。 443 端口一般是用来走 SSL 的，由于 XAMPP 只是本地测试用，SSL 关掉就是了： 1. 打开路径 path-to-xampp\apache\conf\httpd.conf； 2. 搜索 ssl ，大概在第 125 行，把那行代码注释掉，以下是修改后的：

#LoadModule ssl_module modules/mod_ssl.so

3. Apache 可以打开了 Apache 打开了，继续苦逼地工作吧~

我对医生的几点看法

发表于 2012-05-10 更新于 2020-04-19 分类于 Blabla Disqus：

关于医患纠纷暂时想到这么多，遗漏之处日后补充： 1. 现代中国社会，各行各业都只想搞钱，医生也不能例外，这可以理解，不过医生的性质与其他行业不同，医生搞你的鬼你很难知道，可是人的生命是无价的，不应该用作敛财。烦请希望搞大钱的医生们稍微注意一点节操，发扬劫富济贫的风格，别人家很穷又快死了，不该开的药和不该做的检查，能不能还给更有经济承受能力的暴发户去做？ 2. 说“你不相信医生，是不是病了就不看病”，跟说“你觉得水有污染，就不要喝水”，是一样的； 3. 现有体制下，医生不出去乱搞医药代表，我们就可以认为他是一个对家庭负责任的人了。所以，不要把大陆的医生看成多么高尚的工作，你只是做着这个灰色利润丰厚的肥差的一个人而已； 4. 好多人说有些医生工资也很低，但是你要想，凡事都要从低做到高的，即使是抢劫，还得从把风干起不是吗？ 5. 我不怀疑大部分医生都是怀着崇高的理想进入这个行业的，但是你的理想跟社会现实是两码事。就如同，很多人当官之前都想造福于人民一样。所以，说你做过的，而不是想过的； 6. 好多人说，国家政策就是这样，你叫医生不搞灰色收入我怎么生活，医生也是人，也要结婚买房生孩子… 我觉得这些话都有道理。不过，黑奶厂因为利润不足就掺三聚氰胺、用皮鞋做的明胶，黑餐馆因为利润不足就是用地沟油，黑菜贩因为利润不足就使用有毒的保鲜方法，贪官以为入不敷出就挪用公款，这些就是对的吗？难道老百姓连骂一声的权利都没有了吗？ 7. 医生永远是我认为的这个世界上最崇高的职业之一，不包含中国大陆的。

猎豹你好，猎豹再见

发表于 2012-05-02 更新于 2020-04-19 分类于 Web Disqus：

今天又一个浏览器发布了，一看雷军系的，不由得虎躯一震，看了下名字，叫猎豹浏览器，金山公司出品的。有几个视频，意思就说这浏览器对 Webkit 做了百十来项微创新神马的，又能自动切换为 IE 内核啥的，看来应该是来跟 360 极速浏览器抢饭碗的。 虽然我一看就不想下载了，不过感觉牛逼哄哄的样子。后来群里面的人给了个下载地址，才知道这玩意儿也学小米手机搞饥渴营销，一般人还别想安装，于是下载( http://dl.liebao.cn/liebao/KSbrowser_1.0.0.2096_r2.exe )了一个。 安装包足有 30.5M，好奢华的样子。双击打开，就弹出了一个验证码的申请，我扑通就给跪下了，果然是高富帅专用啊，就算是下载了安装包，屌丝也是很难使用的： 后来经 Google 搜索，发现直接拿解压软件打开这个 exe 文件就行了，然后这就成了个绿色软件了，内牛满面啊，看来是一个给屌丝留活路的好软件： 打开 LBBrowser 文件夹下的 liebao.exe ，就启动了这个牛逼的浏览器，冷启动速度还是比较快的，打开网页的速度也不赖，看了一下设置，跟 Chrome 界面相同： 打开了三个网站尝试，他们分别是我自己的blog、支付宝和招商银行信用卡的网站。 打开我自己的 Blog 时，页面上下滚动有迟滞的感觉： 支付宝网站尚无问题，只是安全检查未能生效，也许是因为我用了 360，所以冲突了吧，无视，然后打开招行信用卡的网站，Ooooooooooops… 看来这软件还需要继续迭代，我还是继续用我的 Chrome + IE8 好了，果断删除，猎豹再见！
