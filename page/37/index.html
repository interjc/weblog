变形金刚II

发表于 2009-06-22 更新于 2020-04-19 分类于 Fun Disqus：

本来标题不想用罗马字母II来表达2这个意思的，但2这个字确实扎眼，所以换了。 今年有三部电影值得期待(还一部魔兽争霸说是今年上映，不知道靠不靠谱)，2012、哈利波特与混血王子，再有就是六月份唯一的巨片变形金刚II了。 这部电影的小道消息何其多，我就不粘贴了。说点自己的想法： 还是那个导演，那批人，加入了更多CG和特技，花了更多的钱，故事也不是我们小时候看的那些故事了，不过机器人还是那些人，据说会首次加入合体机器人，不知道会不会有我喜欢的机器人恐龙金锁和他发明的合体计算王(本人从不主动看剧透)。 我们这代从81-85年出生的人，什么好事情都没有赶上，一出生就碰上计划生育，搞得没有兄弟姐妹可陪；上小学就碰见课改，集体做了把第一代小白鼠；这段时间家长们还遭遇了下岗潮，很多家长因此无法为我们今后的学习创业积累资本；高三以前没听说过还可以有素质教育，到了高三的时候还是晚了；上了大学碰见了扩招和不分配工作以及大学生被集体贬低；上班几年又碰见史上首次的金融风暴；还遇见了单位不分房和商品房价年年翻番；够惨的。唯一能让咱们骄傲和自豪的就是咱们小时候看的那些动漫和玩过的那些现如今这些90后觉没有机会再看再玩的东西了，所以变形金刚不光是一部动漫，他还承载着我们81-85这一拨人集体的回忆。 说到小时候看过的动漫，能拍成电影的确实不很多，圣斗士的动漫还在艰难的续拍，一时间很难搬上荧幕了我看；天空战记、宇宙骑士这些有拍成电影的潜质，不过影响力偏小；还有蓝宝石之谜以及一个类似的关于潜艇最后变成了飞船的动漫；蓝精灵我只记得波斯猫了，西曼西瑞我也只记得萨拉囧，以上这些貌似只能拍成小众电影了；还有一些女生动漫比如花仙子、神秘花园等等我看拍成文艺片可行，不过有谁会去看呢？还有国产的葫芦娃等，估计也很难。唯独变形金刚，有日美合作的背景，日本人有创意，美国人会拍大片，一拍即合，就炮制成了现在这部电影了，上一部没有让我失望，所以这一部我依然准备顶着深圳的杀人放血价去电影院观赏，希望迈克尔贝不会辜负我的一番苦心。 BTW：变形金刚II会在本年6月24日在全国各大院线上映，喜欢的别错过了，等电驴估计要3-6个月了，自己权衡吧。 PS：忽然想起自己看过的第一部动漫是《小蝌蚪找妈妈》，而且一开始就是个电影了。
