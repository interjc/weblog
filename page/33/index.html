第九区

发表于 2009-09-19 更新于 2020-04-19 分类于 Fun Disqus：

要说近期比较火的电影，除了那部光演员表就让观众震撼的《见过大爷》之外，就是这部第九区了。 这部片子的IMDB评分高达8.5，豆瓣上更是科幻片少有的8.7分，为了赶上这个热闹，也打上那么一瓶酱油，前两天用电驴驼下来一部R5的看了看。看完既没大呼过瘾，也没深思，直接Shift+Delete了，不内涵地说这也算是近年来一部还好的科幻大片吧。 本片的好与不好说的人太多了，一搜一大把，我就不说了，来谈谈本人的一点看法： 1. 该片特点一，制片国家是南非和新西兰，而不是我们熟悉的美国，这种小众元素很能挑起观众打高分的冲动； 2. 该片特点二，新颖的故事背景，往年说到外星人，要么是可爱的小宠物，要么是强大的侵略者，要么鱼肉我们要么被我们鱼肉，反正人类总是同仇敌忾的与外星人作斗争或者交朋友，最后以美帝国主义为头子的人类联盟干掉了外形侵略者/和外星友人成了好朋友成了主旋律。而该片剑走偏逢不走寻常路我的地盘听我的，连事发地点都胆敢不发生在曼哈顿或者华盛顿或者牛顿，直接放在了南非一个鸟不拉屎的地方，俗称第九区(对不起是南非首都，叫啥忘了，布宜诺斯艾利斯？那好象是阿根廷首都..)，意思就是说一个外星采矿船发生了一点故障，误飞到了地球，管事的又挂了，剩下的都是没头脑的外星矿工，然后在地球被当成难民被安置了下来，这些人长期滞留造成了一些社会问题，该片就是讲这个事情形成的悲欢离合吧。怎么样，新颖吧？ 3. 该片特点三，纪录片形式拍摄科幻片，虽说这个不算新颖了，不过之前的克罗夫档案、Rec等都不算特别大的制作； 4. 那个十分庞大的UFO确实十分拉风，我想许多人就是看到这个猜想看这个片子的吧，不过这玩意儿既不会动也不会攻击，甚至还被人类用电锯锯开了，实在是悲剧啊； 5. 该片想说的其实挺多的，也有很多隐喻，不过相关话题说的够多够滥的了，Right of Races这种话题说了几百年了依然得不到解决，人类为了这个更是争斗了数千年也没个结果，所以说了也是白说； 6. 匪夷所思的跨种族幸服务系统，实在让人联想颇多，话说那是十分的囧啊； 7. 最近好片不多，该片也算是让大家有了一点茶余饭后的谈资了，不过最好不要饭中看，特别是饭中有虾的时候。 暂时就说这么多了，这片子那么多人打五星还是有点道理的，下载的话随便搜一下一大坨我就不放了。 最后一用一段该片的FAQ：

打开CHM文档中的链接出现下载问题的解决方案

发表于 2009-09-15 更新于 2020-04-19 分类于 pc ， Tools Disqus：

有些电脑打开CHM部分CHM文档时，无论点击其中的任意链接，都会弹出下载提示窗口，这个问题不是通过点击“属性”修改那个啥权限(抱歉忘记选项的名字了，应该就是最后一项，如果有的话解除锁定)就可以解决的。 在网上搜了又搜，就是没有一个人一个帖子能解决这么一个问题，答案大多数是叫你修复文件关联或者下载破损的hh.exe文件，可是这些都不是我想要的，也解决不了问题。最后无意中在一份chm文档的下载中找到了解决办法.. 其实这个问题的发生十分简单，就是CHM内的XML文件类型未在系统中注册，只要注册了就行了。具体方法就是修改注册表，将以下代码复制粘贴到记事本，保存为 .reg 文件，之后再打开，就OK了。

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.xml]
@=”xmlfile”
“Content Type”=”text/xml”

当然你也可以直接下载这个已经做好的.reg文件，如果你信得过我的话。 Have fun！

在本机安装Shopex

发表于 2009-09-09 更新于 2020-04-19 分类于 code ， Web Disqus：

想装一个Shopex在本机玩玩，不料还没装上就出了问题了，一片乱码，不管你更换成哪种编码都一样，貌似一盆凉水扑面而来。 遂打开浏览器Google之，发现是没有装Zend Optimizer，按教程找过去，发现官方下载还要注册，于是乎随便找了个网站下载了（下载链接那个难找啊，还好是真的而且没有毒，赞..），下载以后直接安装，直接无脑yes+next肯定是不行的，要选择正确的php及apache目录，好了，一切搞定，重启apache，我欣喜的发现，乱码还是那个乱码。 最后我用了方法若干，找到了解决办法，打开php.ini ，Ctrl+F找到[Zend]，将zend_optimizer.enable_loader = 0 中的0改为1，问题解决。 接下来再次打开Shopex安装页面，这个安装就行云流水轻车熟路了，唯一的可以称之为麻烦的东东也就是需要预先添加一个数据库了。 接下来这个Shopex就算是安装好了，要干什么请便吧。
