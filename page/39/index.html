用虚拟机安装Ubuntu 9.04

发表于 2009-05-11 更新于 2020-04-19 分类于 IT Disqus：

Ubuntu的详细安装教程不是没有，我这里只是简要说明一下用虚拟机的安装Ubuntu的方法。 写在动手之前，你首先要明白，Ubuntu不是免费版的Windows，你安装以后也许并不会觉得他就像传说中那么好用(用你的Windows思维来使用的话)；其次，Ubuntu这个潭子水很深，需要有耐心或者有经验的人才能够很享受的去使用它；另外补充说一点，如果是自己随便玩玩就别下那个服务器版本的，老老实实下载桌面版吧，其实功能都是差不多的，只是服务器版本没有GNU界面而已.. 而后是资源准备。你需要一台能上网的在21世纪购买的计算机，一个ubuntu-9.04-desktop-i386.iso文件，一个虚拟机软件(比如VirtualBox)。 //===============以下是安装步骤=================== 1. 首先安装VirtualBox，如果不清楚怎么设置高级选项的话，直接先一路“下一步”就可以了，碰见安装位置驱动的时候直接无脑“仍然继续”，安装完成有可能会要你重启计算机，虽然你不重启也可以开始用虚拟机了，但是还是建议你重启一下。重启完以后你可能会发现自己上不了网了，如果出现这种情况，点击“网上邻居”–“查看网络连接”–将“VirtualBox Host-Only Network”禁用； 2. 打开VirtualBox，点击“新建”按钮创建一个虚拟机，这时候已经是全中文界面了，相信对大家来说这并非难事，就按自己喜好随便建一个吧，全部按照默认设置也行； 3. 新建好以后别先急着运行虚拟机，先配置一下，点击“设置”，在“光驱”选项中勾选“分配光驱”，按箭头所示“注册”你下载好的的ISO虚拟光盘，如下图所示： 4. 先不要管网络的问题，因为安装Ubuntu的时候如果连接上了网络，系统会自动寻找更新，大大影响安装速度和你等待的心情，这个事情等下再做，这时候我们就可以开始正式安装Ubuntu了，点击“开始”来运行你的虚拟电脑，按图索骥： 5. 装到这里系统算是装到机器里面去了，可是好像不能上网，呵呵，虚拟机的网络还没设置呢，当然不能上网，把虚拟机关掉(不是把VirtualBox关掉，怎么关机应该都知道了吧..)，点“设置”，进“网络”选项，其中第二个下拉菜单选择“Bridged Adapter”，就是桥接的意思，具体意义分析来也没啥实际意义，反正这样选了就可以上网了可能大概应该..顺手在“光驱”选项里把“光驱”选项取消掉或者把虚拟光盘释放掉吧，系统都装完了光盘自然也就没用了.. 6. 这时候我们再次打开虚拟电脑，是不是已经可以上网了呢？反正我是可以了。 写到这里这个可以上网的Ubuntu已经安装完成了，别高兴得太早，好戏才刚刚开始，还有plenty of work等着你去做呢，我推荐你如果感兴趣并有时间的话去Ubuntu社区潜心学习吧，路漫漫其修远兮，壮士一去兮不复反，祝您好运！

入手Dell Inspiron 1427

发表于 2009-04-25 更新于 2020-04-19 分类于 琐事 Disqus：

mumu和我都迫切的需要使用电脑来实现或提高自己的价值(有数据为证)，而电脑只有一台，这成为了家里的首要不和谐因素，唯有再添置一台才能解决这个不可调和的阶级矛盾，所以，趁着前几天刚入手BlackBerry 8820的余兴，昨天下午，我们以迅雷不及掩耳盗铃之势将Dell Inspiron 1427抱回了家。配置如下图: 买这个本本之前其实早有预谋，首先在心里盘算了下，本着构建节约型社会的原则，把预算缩减到5K左右，然后定下了ThinkPad、Dell、HP这三个主要目标，遂开始四处寻本。刚开始比较看好ThinkPad SL400，经过多番调查比对，采纳大量群众意见(其中包括zEUS.同学的对于这款机型的强烈鄙视和反对)，终于发现盲目崇拜是不行的，纸老虎也是容易被捅破的，总之这款机子就不予考虑了，Pass；然后是HP的6531s和CQ45等等，HP的机器型号实在是多，实属机海战术，给我的挑选带来了极大的不便，后来通过广泛的调查网络评论，发现此品牌的中低端笔记本普遍存在散热极差的问题，外加HP的笔记本型号实在太多，一个型号下又有N中不同的配置和价格，有点子子孙孙无穷匮也的感觉，所以HP也Pass掉；最后是Dell，其实我对Dell有一种潜在的厌恶之情(以前，因为公司的电脑全是Dell啊。你想想，武汉人买车也不情愿去买个富康吧天津人也不愿意买夏利吧，开出去别人说是的士..)，但是挑来挑去已经Pass得只剩它了，那就挑吧，开始时只把眼光放在Studio和Latitude等系列上，这样要么配置不让我满意要么价格远超预算，由于这款机器主要是mumu用，所以是预算第一、品牌外观稳定售后第二，在此基础上谋求配置最大化就可以了，于是乎开始研究Inspiron系列家用笔记本，这下就豁然开朗了。 一开始定下的是Inspiron 1420，等我们跑去电脑城的时候才发现想现场买Dell的话，这个水也不浅，很多都没有发票不说，也都拿不到现货，好多配件也不知道是不是原装的，实在害怕上当受骗，后来在学府路的苏宁Dell体验店以5199的网上折扣价买下了这台Dell Inspiron 1427(CPU：T4200 内存：2G 硬盘：250G 独显：9300M 512M)，还送一鼠标，在商场购物就是放心的多，连送的那个鼠标都有发票..开箱验机发现系统是Vista Home Basic SP1的，销售人员不给换XP，说是要等15天，而且要换也没有正版OEM的还不如我自己换，于是开开心心拿回家，用前几天买BB的时候顺手拿回来的TP-Link 340G+上网，速度很稳定。顺带说一下，Vista的安全设置过于复杂且不说了，反正是要换的，赠送的那个麦咖啡杀毒软件着实恶心，速度慢且不说还老是弹出各种无聊的警告窗口，马上换成NOD32，世界一下子就安静了。现在坐等15天以后装这个我刚下载的原装Dell OEM版XP SP3。 给几个图吧，我刚用我的A720照的，本人近照欠奉..
