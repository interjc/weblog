在搜索引擎上求职和招聘

发表于 2008-03-11 更新于 2020-04-19 分类于 idea ， Web Disqus：

本人一直认为，现如今能搞创造收益的网络服务一定要与人民群众生活迫切相关才可以达到效果。现在电视上广告上多如牛毛的就是求职网站的广告，从某种程度上应证了我的想法。 每个人想要找工作、跳槽、招人的时候，能想到的除了人山人海的招聘会就是网络了。招聘会的好处是场面火爆，而且多数是投简历和面试二合一，缺点显而易见，你除了要学会怎样大浪淘沙以及人肉屏蔽诸如北大青鸟培训之类的伪招聘之外，更要学会如何照看好你的随身贵重物品，并且对于想跳槽的同志来说，在招聘现场碰见现单位同事甚至上司的情况也并不是不可能出现的。所以网络求职展现了他的长处。网络求职更加经济实惠、节约时间，而且目前很多网站发明的批投简历功能实在令人乍舌，不可以简单的用牛×来形容。 说了这么多无非想说明，网络求职是很有利可图的，所以才会有了多如牛毛的求职网站。大大小小的求职网站，你可还一个都别怠慢，因为各网站都有自己的势力范围和人脉，有时候会通过各种手段保持自己网站上招聘资源的唯一性，像中华英才网、前程无忧、智联招聘这样的网站估计是一个都不要少比较好，还有各省、各市、各区各地方的求职网站你也最好去看看(个别极端才华横溢酒好不怕巷子深从小立志只进全球二十强只等各CEO屡顾茅庐的达人可忽略前面所述)。这么多网站，你需要填写多少资料、注册多少ID、进行多少次搜索呢？而且各网站搜索性能良莠不齐，有些甚至只有多达几百页的列表，难道你真的希望每天投上几千份简历然后在家坐等手机被打爆？ 近段时间看见一个电视广告，是一个求职网站的，大意说是：您求职碰壁了吧，这么多碰壁干嘛不换一家。是啊，可是你换了一家难道就不会碰壁了？这时候我想到了搜索引擎。现如今流行广泛的无非就是两大搜索引擎，我用“南山区 深圳 招聘 网络工程师”这个关键词随便试了一下，在Google.cn和百度得出的结果都不甚如意，百度上赫然三条置顶广告并且都是培训的，其他结果中大多数也是如此。 为什么做人一定要这么短视呢，直接把搜索开辟成为求职搜索平台不好吗？或者直接建一个子站叫谷歌求职或者百度打工，不好吗？如果依托自己如吸星大法一般的网络信息收录量和精确的搜索运算和强大的市场统计能力，由搜索引擎建立的求职信息平台绝对所向披靡。搜索引擎可以出视频搜索，弄出个求职搜索是难事吗？由此衍生的商业模式也即是赚钱手段难道还少吗？并且口碑还不会坏。 PS：匆忙之中憋出这么一篇，十分粗糙，算是抛砖引玉吧，欲知后事如何且听下回分解，毕。
