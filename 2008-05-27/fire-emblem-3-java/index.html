手游:火焰纹章III三部曲

发表于 2008-05-27 更新于 2020-04-19 分类于 Fun Disqus：

有读者说我好久不更新了，我也实在是不好意思，最近比较懒，外加又把《金婚》重新看了一遍，再加上国情实在是让人悲恸，不大想更新。可生活总是要继续吧，于是我就更新一篇吧，是个游戏，爱国同仁您也别急着骂娘，我这也是为了调节国人紧张的思想状态嘛.. 火焰纹章是个什么东东？看看他们的民间官网就知道了，这个游戏是一个比较经典的战棋类史诗游戏，基本上什么游戏机平台都经历到了，由于是其他平台移植过来的，所以不要担心他的趣味性，这个游戏和其他那些玩一个小时就穿了的小玩意是不同的。玩这游戏颇费精神，当然也很耐玩。 废话我就不说了，借几张图吧..攻略什么的您自己搜吧.. ))))) 放心，这个游戏是java的，也就是说，基本上所有手机都可以玩了，屏幕大小也是自适应的，不用考虑机型问题，至于破解程度，我试了一下，基本上除了终章都完美破解了，没有收费，如果信不过可以打开飞行模式(如果你有这个模式的话..)玩，通过测试再调回来。 游戏分三个部分，初章、二章和终章，终章由于是新出的，并没有完美破解，如果出了，我会补上的..附送简单攻略几个，外加直接通关秘笈一枚(回复里有人问我我就说..呵呵..)。一起打包了，反正也不大，解压zip文件和往手机里安装java游戏就不用我教了吧，实在不懂自己去百度知道送分吧..下载地址如下： 打包下载：纳米盘、Box.net PS：仅供测试用，如果你喜欢这个游戏请删除再去下载正版，本人不对您下载所产生的任何问题负责..
